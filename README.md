# AI-Agent-Chat-Platform
แพลตฟอร์มแชท AI ที่สามารถเชื่อมต่อกับผู้ให้บริการ AI หลายราย (Multi-API / Multi-LLM) เช่น OpenAI, Gemeni หรือ API อื่น ๆ โดยผู้ใช้สามารถโต้ตอบผ่านหน้าแชท พร้อมมีระบบจัดเก็บประวัติการสนทนา

## 🚀 Features (ความสามารถหลัก)

- 💬 ระบบแชท AI แบบ Real-time
- 🔌 รองรับการเชื่อมต่อหลาย API (Multi-Provider LLM)
- 🔑 ระบบจัดการ API Key ของผู้ใช้
- 🧠 รองรับการเลือก Model / AI ที่ใช้งาน
- 🗂️ บันทึกประวัติการสนทนา (Chat History)
- 👤 ระบบจัดการผู้ใช้ (User Session / Authentication)
- ⚡ ออกแบบให้สามารถขยายระบบได้ (Scalable Architecture)
